PARA AJUDAR NA SUA PESQUISA, REUNIMOS A MAIOR COLEÇÃO DAS DIFERENTES LINGUAGENS TÉCNICAS DA INTERNET:

HTML

  • É a linguagem de marcação padrão para a criação de páginas web;
  • É constituído de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria;
  • Consiste em criar um conjunto de regras que estruturam componentes visuais.

CSS

  • É a linguagem que usamos para estilizar um documento HTML;
  • Descreve como os elementos HTML devem ser exibidos em páginas web;
  • Pode controlar o layout de várias páginas web de uma só vez.

JavaScript

  • É a linguagem de programação usada para desenvolvimento Web;
  • Permite a implementação de itens complexos em páginas Web;
  • É uma das três principais tecnologias da World Wide Web.

PHP

  • É uma ferramenta poderosa para criar páginas Web dinâmicas e interativas;
  • É uma alternativa amplamente utilizada, gratuita e eficiente;
  • Pode ser utilizado na maioria dos sistemas operacionais.

Python

  • Pode ser usado em um servidor para criar aplicativos da web;
  • Pode ser usado junto com o software para criar fluxos de trabalho;
  • Tem uma sintaxe simples semelhante à da língua inglesa.

React Native

  • É usada para desenvolver aplicativos de forma nativa;
  • É compatível com os sistemas operacionais IOS e Android;
  • Criado pelo Facebook em 2015 sobre a licença MIT.

Bootstrap

  • O Bootstrap é a estrutura HTML, CSS e JavaScript mais popular para o desenvolvimento de sites responsivos que priorizam dispositivos móveis;
  • O Bootstrap é totalmente gratuito para baixar e usar.

p5.js

  • O p5.js é uma biblioteca JavaScript para programação criativa e inclusiva para artistas, designers, iniciantes e qualquer outra pessoa;
  • O p5.js é gratuito e de código aberto porque acreditamos que o software e as ferramentas para aprendê-lo devem ser acessíveis a todos.
